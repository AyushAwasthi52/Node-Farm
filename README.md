# Node Farm

A full-stack project built using pure Node.js without Express.js to understand how backend systems work at a lower level.

This project focuses on learning how servers, routing, request handling, form processing, and frontend-backend communication work internally without relying on backend frameworks like Express.

The application handles form submissions and demonstrates how raw Node.js can be used to build backend functionality from scratch.

---

## Features

- Backend built without Express.js
- Native Node.js HTTP server
- Form submission handling
- Frontend and backend integration
- Manual routing system
- Request and response handling
- Basic server-side architecture
- Static file serving

## How It Works

```text
User Fills Form
        ↓
Form Data Sent to Node Server
        ↓
Server Processes Request
        ↓
Response Returned to Client
        ↓
Frontend Updates
```

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Node-Farm.git
```

Move into the project directory:

```bash
cd Node-Farm
```

Install dependencies (if any):

```bash
npm install
```

Run the server:

```bash
node index.js
```
